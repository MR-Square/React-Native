* How can we create a list in RN?
    To create a list in RN we use "FlatList" widget.

* Some important attributes of FlatList widget:
    * data  : An array (or array-like list) of items to render.
    * renderItem    : Takes an item from data and renders it into the list.
    * keyExtractor  : Used to extract a unique key for a given item at the specified index.
